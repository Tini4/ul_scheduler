# UL scheduler

A browser extension to help with scheduling at the University of Ljubljana.

## Supported faculties

- BF
- FMF
- FRI
- FS

## Features

- GUI fixes
- Entry removal
- Undo with Ctrl+Z
- Link generation (FRI)
- Adding entries
- **New** in 1.1.x: Schedule saving

## Warning

Modification of the schedule is done in your browser.
If you close or refresh the page, any unsaved changes will be lost.
